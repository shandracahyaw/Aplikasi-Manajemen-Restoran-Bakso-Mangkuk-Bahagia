# Aplikasi-Manajemen-Restoran-Bakso-Mangkuk-Bahagia
Repository ini berisi implementasi proyek "Manajemen Restoran" menggunakan bahasa pemrograman Java, sebagai bagian dari Tugas 3 Matakuliah Berpraktik Pemrograman Berbasis Desktop (PBD) di Universitas Terbuka. Proyek ini dirancang untuk mengaplikasikan berbagai konsep dasar dan lanjutan dari OOP.
